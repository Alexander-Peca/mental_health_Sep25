# Mental Health Project (Sep 2025)
Mental Health Literacy Intervention Analysis

Objective:
Evaluate whether a ChatGPT-based intervention improves mental health literacy and related psychological outcomes using a pre/post study design.

Methods:
- Data cleaning
- Exploratory data analysis
- Paired t-tests
- Effect size estimation (Cohen's dz)
- Confidence intervals
- Assumption diagnostics

Main findings:
- Significant improvement in Mental Health Literacy (MHL Total)
- Significant improvements in several MHL subdimensions
- Significant improvement in Mental Health Self-Efficacy
- Positive participant evaluations of the intervention
